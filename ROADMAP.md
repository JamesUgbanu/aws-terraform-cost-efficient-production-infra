# Roadmap

This roadmap is intentionally practical. Each item is a good candidate for a
GitHub issue or small public contribution.

## Next Improvements

- Add ACM and Route53 modules
- Add blue-green ECS deployment example
- Add optional Lambda job runner module
- Add first-class SQS worker pattern module
- Add private subnet mode with VPC endpoints enabled
- Add fuller NestJS deployment example
- Add remote state bootstrap for S3 and DynamoDB
- Add optional ElastiCache module for caching-heavy workloads
